HKU 2026 Summer Workshop on Statistics and Data Analytics

This repository contains the source files for the HKU 2026 Summer Workshop on Statistics and Data Analytics, organized by the Statistics Group within the Innovation and Information Management Area of HKU Business School.

The website is a simple static site written in HTML and CSS. It provides basic information about the workshop, including invited speakers, program, venue, transportation and accommodation.

To update the site content, edit the HTML files in the root directory and the subfolders such as sessions, venue and Transportation, and the shared stylesheet in the assets folder.

You can deploy this site on any static web server by uploading the files in this repository to your hosting service.
